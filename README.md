# Markdown Snippets for Zed

## Installation

You can install this directly on the [Zed extensions
page](https://zed.dev/extensions/markdown-snippets) or by following these steps:

1. Clone this repo:

   ```sh
   git clone https://github.com/bobbymannino/markdown-snippets-for-zed
   ```

2. Go to Extensions menu in Zed IDE
3. Click "Install Dev Extension"
4. Select the folder you cloned

## Available Snippets

### Markdown

- `gh-info`: A GitHub flavored note alert
- `gh-tip`: A GitHub flavored tip alert
- `gh-important`: A GitHub flavored important alert
- `gh-warning`: A GitHub flavored warning alert
- `gh-caution`: A GitHub flavored caution alert
- `cmt`: A comment
- `link`: A hyperlink
- `img`: An image
- `code`: A code block
- `code-css`: A CSS code block
- `code-ts`: A TypeScript code block
- `code-js`: A JavaScript code block
- `code-jsx`: A JSX code block
- `code-tsx`: A TSX code block
- `code-bash`: A Bash code block
- `code-yaml`: A YAML code block
- `code-sql`: A SQL code block
- `code-json`: A JSON code block
- `code-html`: An HTML code block
- `code-md`: A Markdown code block
- `code-py`: A Python code block
- `code-rs`: A Rust code block
- `code-go`: A Go code block
- `code-toml`: A TOML code block
